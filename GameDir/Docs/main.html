 This is the Doc's for LieroXtreme. There is a menu to the left side of the screen which have sections that can pretty much explain everything about the game.

About
Liero Xtreme is a clone/sequel of the classic freeware game Liero.

The aim is to create a game like Liero, but with a few changes/additions:

  • ·Higher resolution
    Liero Xtreme uses 640x480, but the map and other main game graphics are stretched to give the classic 320x200 feel to the game.
    The higher res allows for a better user interface for the main menu, game setup,etc screens
  • ·Integrated Map Editor
    The original Liero had a map editor that was a seperate program. Liero Xtreme has a built-in map editor that allows the players to create the map before they play, or preview the map that they are about to do battle on.
  • ·Moddability
    Liero Xtreme is designed to read all the weapon data from a simple script. In fact, all the weapons currently in the game are all from a script. None are hard coded in.
  • ·More AI
    Liero Xtreme has the ability for you to play against up to 7 computer AI opponents. Some say the AI is smarter.
    ·Net' Play
    Liero Xtreme has network feature which means you can actually verse other players from around the world. Up to 8 online players can play simultaneously as you battle it out for the winner! Liero Xtreme even gives you the opportunity to host you very own server off your own computer.
